# Saudi Radio 📻

A modern, user-friendly iOS app that allows users to stream all Saudi radio stations. Built with SwiftUI, this app provides a clean interface for enjoying Saudi Arabia's radio broadcasts anytime, anywhere - even when the device is locked.

![Saudi Radio App](https://github.com/alshehri12/SaudiRadio/raw/main/screenshots/app-preview.png)

## Features ✨

- **25+ Saudi Radio Stations**: Access to official Saudi Broadcasting Authority stations and popular private channels
- **Beautiful UI**: Clean, intuitive interface inspired by modern streaming apps
- **Categories**: Browse stations by type (News, Quran, Music, Talk, Sports, Entertainment)
- **Favorites**: Save your preferred stations for quick access
- **Search**: Easily find stations by name
- **Background Playback**: Continue listening even when the app is in the background or the screen is locked
- **Lock Screen Controls**: Control playback directly from your lock screen

## Stations Included 📡

- **General Radio Stations**: Saudi Radio, Riyadh, Jeddah, Makkah, Madinah
- **Quran & Islamic**: Saudi Quran, Sunnah Channel, Nidae AlIslam
- **Regional Stations**: Dammam, Hail, Qassim, Abha, Jizan
- **Entertainment & Music**: MBC FM, Rotana FM, Panorama FM, UFM, Mix FM Saudi
- **News & Sports**: Al-Arabiya FM, Saudi Sport, Saudi International

## Technical Details 🛠️

- **Framework**: SwiftUI
- **Architecture**: MVVM (Model-View-ViewModel)
- **Minimum iOS Version**: iOS 15.0
- **Audio Streaming**: AVFoundation with background mode capability
- **State Management**: Observable objects with Combine framework
- **Persistence**: UserDefaults for favorites storage

## Installation 📲

1. Clone this repository
2. Open `SaudiRadio.xcodeproj` in Xcode
3. Select your target device and build the app

## Requirements 📋

- iOS 15.0+
- Xcode 14.0+
- Swift 5.7+
